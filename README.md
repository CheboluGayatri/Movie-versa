# AI Movie Verse

AI Movie Verse is a responsive web application that lets users explore movies by category and watch their songs on YouTube. It also features a simple AI-inspired suggestion system that recommends movies based on user mood.

# Features

Movie Categories: Love, Action, Comedy, Sci-Fi

Movie Cards: Posters, ratings, and direct YouTube song links

AI Movie Suggestion: Suggests movies based on mood input

Responsive Design: Works on desktop and mobile devices

Easy to Extend: Ready for API integration or additional movie data

# Technologies Used

HTML5

CSS3

JavaScript

How to Use

Open the live site or download the repository.

Browse movies by category.

Click Watch Songs to view songs on YouTube.

Use the AI Movie Suggestion input to get movie recommendations based on your mood.
